# Manual de publicação em redes sociais

Este manual explica como publicar manualmente os criativos gerados pelo Estúdio Criativo enquanto as contas oficiais das redes sociais ainda não estão conectadas para publicação automática.

## Quando usar este manual

Use este fluxo quando a publicação automática agendada ainda não estiver ativa. A automação exige conexão oficial de cada rede, como Instagram/Facebook via Meta Graph API, LinkedIn API, TikTok for Business, YouTube Data API, X API e demais provedores compatíveis.

## Fluxo recomendado

1. Acesse o app e entre no Estúdio Criativo.
2. Gere ou envie a imagem base do criativo.
3. Aguarde a criação das variações oficiais para redes sociais.
4. Clique em **Baixar** no formato desejado ou em **Baixar pack completo (.zip)**.
5. Extraia o arquivo `.zip` no computador.
6. Abra a rede social desejada.
7. Crie a publicação manualmente usando o arquivo correspondente ao formato da rede.
8. Copie a legenda e hashtags planejadas no agendamento do app, se houver.
9. Publique imediatamente ou use o agendador nativo da própria rede.

## Formatos gerados pelo Estúdio Criativo

O pack pode conter variações para Instagram, Facebook, TikTok, LinkedIn, X, YouTube, Pinterest e WhatsApp. Sempre escolha o arquivo cujo nome corresponde à rede e ao posicionamento desejado, como feed, story, reels, capa, thumbnail ou post quadrado.

## Publicação automática

A fila de agendamento no app registra data, legenda, hashtags, criativo e redes selecionadas. Para que a postagem saia automaticamente, é necessário conectar as contas oficiais de cada rede via Lovable Cloud.

Enquanto essas conexões não estiverem ativas, o agendamento serve como planejamento interno e o arquivo deve ser publicado manualmente seguindo o fluxo acima.

## Checklist antes de publicar

- Confirme se a arte está no tamanho correto da rede.
- Revise textos, preços, datas e informações legais.
- Verifique se a legenda contém CTA e hashtags corretas.
- Teste a visualização da imagem na rede antes de confirmar.
- Guarde o `.zip` original como backup da campanha.
